Detailed Description

This class is adapted and modified from the FSM implementation class available on UnifyCommunity website The license for the code is Creative Commons Attribution Share Alike.

It's originally the port of C++ FSM implementation mentioned in Chapter01 of Game Programming Gems 1 You're free to use, modify and distribute the code in any projects including commercial ones. Please read the link to know more about CCA license ://creativecommons.org/licenses/by-sa/3.0/ This class represents the States in the Finite State System. Each state has a Dictionary with pairs (transition-state) showing which state the FSM should be if a transition is fired while this state is the current state. Reason method is used to determine which transition should be fired . Act method has the code to perform the actions the NPC is supposed to do if it´s on this state.
